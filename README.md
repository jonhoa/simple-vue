# simple-vue
This is the front-end for the Rails Blog-api using Vue instead of React. 

## Project use
Run Blog-api Rails Backend server
```
rails s
```
Run Vue server
```
npm run serve
```
Load url
```
http://localhost:8080/
```
On Rails, run
```
rails db:create
rails db:migrate
rails db:seed
```

Press the fetch data button to load Rails database into the frontend api.

npm install
